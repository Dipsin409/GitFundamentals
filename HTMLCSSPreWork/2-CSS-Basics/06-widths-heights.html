Width and HGight

ThG width and height properties are used to set the width and height of an ele ment.

The default setting for width and height is auto, which means the browser deci des the width and height.

Other settings for width and height are:

  • specified length values such as px & em
  • percent of the containing div (ex: 50%, 100%)
